# is-letter

[![npm version](https://badge.fury.io/js/is-letter.svg)](https://badge.fury.io/js/is-letter)

> It is a letter?

## Install 

```
npm install is-letter
```

## Usage

```js
var isLetter = require('is-letter');
isLetter('a'); // => true
isLetter('abba'); // => false
```
